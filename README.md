# Spotify Music Data Analysis using Pandas

## Project Overview

This project focuses on analyzing Spotify music data using Python and Pandas. The analysis helps discover insights related to songs, artists, genres, albums, and audio features through data cleaning and exploratory data analysis techniques.

---

# Objectives

* Clean and preprocess Spotify dataset
* Analyze song popularity trends
* Identify top artists and albums
* Perform genre-based analysis
* Explore relationships between audio features
* Generate insights using Pandas

---

# Technologies Used

* Python
* Pandas
* NumPy
* Jupyter Notebook

---

# Dataset Information

The dataset contains Spotify music-related information such as:

* Track Name
* Artist Name
* Album Name
* Playlist Genre
* Popularity
* Danceability
* Energy
* Loudness
* Speechiness
* Acousticness
* Instrumentalness
* Valence
* Tempo
* Release Date

# Data Cleaning

The dataset was cleaned and prepared for analysis by:

* Removing unnecessary columns
* Handling missing values
* Removing duplicate records
* Renaming columns
* Checking data types
* Converting release date into datetime format

# Key Insights

* Energetic songs are generally louder.
* Danceable tracks tend to have higher popularity.
* Some artists dominate the dataset with a large number of tracks.
* Acoustic songs are usually less energetic.
* Popular genres have higher average popularity scores.
